# 🛍️ Mini E-Commerce Website

A simple and responsive shopping website built using HTML, CSS and JavaScript.

## Features

* Homepage
* Product listing
* Shopping cart
* Quantity update
* Checkout page
* Order success page
* Responsive design

## Technologies Used

* HTML
* CSS
* JavaScript

## Project Structure

* index.html – Homepage
* products.html – Product page
* cart.html – Shopping cart
* checkout.html – Checkout page
* success.html – Order success page

## Purpose

This project was created to practice front-end web development and build a mini e-commerce shopping experience.
